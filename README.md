# Unemployment Analysis with Python

## CodeAlpha Data Science Internship - Task 2

### Project Overview

This project analyzes unemployment trends across India using Python. The analysis focuses on understanding unemployment patterns, regional variations, labour participation rates, employment estimates, and the impact of COVID-19 on unemployment.

### Objectives

* Clean and preprocess unemployment data.
* Perform exploratory data analysis (EDA).
* Visualize unemployment trends across states and regions.
* Analyze the impact of COVID-19 on unemployment rates.
* Identify states with the highest and lowest unemployment rates.
* Generate insights for policy and economic understanding.

### Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

### Project Workflow

1. Data Collection
2. Data Cleaning
3. Missing Value Handling
4. Exploratory Data Analysis
5. Data Visualization
6. COVID-19 Impact Analysis
7. State-wise and Region-wise Comparison
8. Insight Generation

### Key Analysis Performed

* Correlation Heatmap
* Distribution of Unemployment Rate
* State-wise Unemployment Analysis
* Region-wise Analysis
* Labour Participation Rate Analysis
* Employment Estimation Analysis
* Monthly Trend Analysis
* COVID-19 Impact Analysis
* Top 10 Highest Unemployment States
* Top 5 Highest and Lowest Unemployment States

### Results

* Significant variations in unemployment rates were observed across states.
* COVID-19 had a noticeable impact on unemployment levels.
* Certain regions experienced higher unemployment than others.
* Labour participation rates differed significantly across states.
